How to add your signature in Outlook:
1. Click Copy HTML above
2. Open Outlook → Settings → Signatures → New
3. In the signature editor, press Ctrl+Shift+X to open the HTML editor
4. Paste the HTML code and save

How to add in Gmail:
1. Open Gmail → Settings → See all settings → Signature
2. Click into the signature box, press Ctrl+Shift+X to open the HTML editor
3. Paste the HTML code and save
